These patches are my sprite edits for the GBA games Boktai 1 & 2, which change Django's hair sprite to match the much nicer and representative style that features in the Boktai 3.

These patches only change the hair, not the rest of the body. That may be something to work on another time! But for now, please enjoy this simple little sprite mod!

Note that in Boktai 1 this will also change Sabata's hair too, as he seems to share the sprite, just with a pallete swap. Sabata will keep his own hair in Boktai 2, as he has his own updated sprite that is kept seperately.

---

**Click on the patch you would like above, and then find the download button.** To use the patch, you will need to use a patching tool to apply them to a ROM, which you should be able to find online. The one I have used previously is Lunar IPS but I imagine there are others available online to choose from.

These patches we're also tested with a solar sensor patch that is commonly used to control the light gauge. I would reccomend applying this sprite patch to a clean ROM first, and then using one of these solar sensor patches afterwards.
